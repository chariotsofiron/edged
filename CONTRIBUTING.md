
# Linting

```bash
cargo clippy -- -W clippy::all -W clippy::pedantic -W clippy::nursery -W clippy::restriction -A clippy::mod_module_files -A clippy::implicit-return -A clippy::missing-inline-in-public-items -A clippy::indexing-slicing -A clippy::missing_trait_methods


-A clippy::std-instead-of-core -A clippy::single-char-lifetime-names -A clippy::std-instead-of-alloc -A clippy::multiple-inherent-impl
```
